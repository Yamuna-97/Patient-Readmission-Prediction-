## 📌 Project Overview

<div align="center">

🧠 This project predicts whether a patient will be **readmitted to the hospital** after discharge.

📊 It uses machine learning models trained on healthcare data to identify **high-risk patients early**.

💡 Helps hospitals improve patient care and reduce unnecessary costs.

</div>

---

## 🎯 Objective

* Predict patient readmission risk
* Assist healthcare decision-making
* Reduce hospital readmission rates
* Improve patient outcomes

## ⚙️ Installation & Setup

```bash
git clone https://github.com/Yamuna-97/Patient-Readmission-Prediction
cd Patient-Readmission-Prediction
pip install -r requirements.txt
streamlit run app.py
```

---

## 🚀 How It Works

1. Enter patient details in the web app
2. Click on **Predict**
3. Model analyzes input data
4. Displays prediction result (Readmitted / Not Readmitted)

---

## 📊 Model Insights

* Uses structured healthcare data
* Handles missing values and encoding
* Learns patterns from patient history
* Provides fast prediction output
